# v0.2.0 - TAH Command Center

Date: 2026-06-16

## Summary

This release turns Sunset Pulse into a practical real estate agent command center powered by TAH-based local intelligence.

The main shift is architectural: instead of treating AI as one general chat box, the app now routes agent commands through specialized workers, retrieves compact `.tah` context, renders the answer with a selected relay template, and stores useful query memory locally for future token savings.

## Product Thesis

Sunset Pulse does not make agents chat with AI. It lets agents command their own private market intelligence layer.

## Main Features

### Command Center

- New `/command-center` route.
- Professional worker selection screen inspired by character-select layouts.
- Auto and manual worker routing.
- Supervisor check toggle.
- Developer tools panel with command-post trace data.

### TAH Worker Loadouts

Added first-party TAH capsules:

- `agent_brand.tah`
- `lead_history.tah`
- `listing_context.tah`
- `neighborhood_context.tah`
- `comps_context.tah`
- `objection_scripts.tah`
- `local_business_context.tah`
- `market_rules.tah`

These replace virtual placeholders with real local context files.

### Segmented Expert Atlas

- 400-expert cap.
- Concept-anchored segmentation.
- Density and vitality scoring.
- Recursive binary concept linking between shards.
- Loadout-aware retrieval pass.
- Domain and source gating for command-center relevance.

### Relay Templates

Added a 68-template relay catalog for explaining TAH findings.

Delivery formats:

- Briefing Card
- Slideshow Deck
- Puppetshow Explainer
- Field Board
- Talk Track Script

Every relay includes a final provenance screen showing:

- where the information came from,
- which `.tah` files were used,
- concepts retrieved,
- what the user learned.

### Local Query Memory

Sunset Pulse now appends compact local memory records to:

```text
apps/pulse/cartridges/query_memory.tah
```

The command router recalls matching prior query memories before new retrieval so repeated work can reuse context and save tokens.

This memory file is local-only and ignored by Git.

## API Surface

```text
GET  /api/commands
POST /api/commands
GET  /api/tah/fact
```

`POST /api/commands` accepts:

```ts
{
  command: string;
  selectedWorkerId?: string;
  relayMode?: 'briefing' | 'slideshow' | 'puppetshow' | 'field-board' | 'script';
  supervisor?: boolean;
}
```
