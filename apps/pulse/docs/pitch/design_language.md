# Sunset Pulse // Design Language & Pitch Components
## Visual Identity for the "Premium Tech" Aesthetic

### 1. Color Palette: "The Midnight Horizon"
To build trust and convey technical sophistication, we use a high-contrast, dark-mode-first palette.
- **Primary (Deep Navy):** `#0A192F` (Deep space, used for backgrounds and depth).
- **Secondary (Slate Gray):** `#2D3748` (Neutral balance for cards and containers).
- **Accent 1 (Sunset Orange):** `#FF8C42` (Warm highlights, "Pulse" indicators).
- **Accent 2 (Pulse Purple):** `#7B2CBF` (Intelligence layers, AI activity).
- **Text (Off-White):** `#E2E8F0` (High readability against dark backgrounds).

### 2. Typography: "Clean Intelligence"
- **Headings:** **Inter** (Bold/Extra-Bold) - Geometric, modern, and authoritative.
- **Body:** **Roboto** (Regular/Medium) - Optimized for legibility across screen sizes.
- **Monospace (Data/Code):** **JetBrains Mono** - Used for highlighting technical specifications and "Tech Stack" validation.

### 3. Core Component Checklist (Slide Layouts)
Each slide must adhere to the "3D First" layout principles, using edge-to-edge containers.

| Slide Type | Focus Area | Visual Strategy |
| :--- | :--- | :--- |
| **The Hook** | Industry Friction | Grayscale, high-noise imagery of legacy portals vs. clean Sunset Pulse UI. |
| **The "Pulse" View** | 3D Spatial Search | A hero layout featuring a high-res render of the Three.js property interface. |
| **Jamie AI Deep Dive** | Intelligence Layer | Tactical green overlays, "scanline" effects, and "Council of Judges" avatars. |
| **Market Integration** | IDX/NTREIS Data | A clean architectural diagram showing the seamless data flow from Repliers.io to Supabase. |
| **The Roadmap** | Future Velocity | A linear timeline slide with "Sunset" gradient accents for future feature milestones. |
| **The Visionary** | Closing Statement | A minimal, high-impact slide with the core mission statement and founder contact. |

### 4. Interactive Elements (Pitch Prototype)
For the live pitch, use embedded video loops or a "Mini-Pulse" React component to demonstrate:
- **Reactive Physics:** Showing the `<TacticalCloth />` reacting to the presenter's movements (if using a camera-enabled demo).
- **Real-Time Data:** A simulated live feed of "Neighborhood Recon" alerts.
