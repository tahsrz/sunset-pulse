# 🎨 Jamie Agentic UI: Styling Hooks

This document outlines the "Styling Hooks" available for the Sunset Pulse Command Post. You can ask Jamie for these specific looks, or use them as a basis for your own custom requests.

## 🚀 How to Use
While in **Dev Mode** (toggle the terminal icon in Jamie's chat), ask Jamie things like:
- *"Jamie, go dark mode."*
- *"Jamie, make the dashboard look like a cyberpunk terminal."*
- *"Jamie, I want a moody, high-contrast dashboard."*
- *"Jamie, clean up the UI, make it minimalist."*
